Show multiple pages on a page.

== Description ==

This plugin can display more than one page on a page.

== Installation ==

1. Upload `showmultiplepages.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<!--pageid=##-->` or `<!--pagename=???-->` in your sites.

== Frequently Asked Questions ==

= How can i display multiple pages on one page? =

Insert `<!--pageid=##-->` or `<!--pagename=???-->` in your sites. Every pages you defined in the page, will be displayed when you open the page in your browser.
